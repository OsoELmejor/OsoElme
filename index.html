Hoy en dia, Javascript es el unico lenguaje capaz de ejecutarse
en las 3 capas de una aplicacion:

  • Frontend (JavaScript)
  • Backend (Node.js)
  • Persistencia de Datos (MongoDB, Couch DB, Firebase, etc).

  • Experiencia 3D, Realidad Aumentada, Realidad Virtual.

  • etc.

Javscript NO es Java

Orígenes y Propósito: Java fue desarrollado por Sun Microsystems en la década de
1990 como un lenguaje de programación de propósito general orientado a objetos.
Fue diseñado para ser utilizado en una amplia gama de aplicaciones,
desde aplicaciones de escritorio hasta sistemas distribuidos en red. Por otro lado,
JavaScript fue creado por Netscape en la misma época, pero con un propósito
diferente: mejorar la interactividad en las páginas web. Tipo de Lenguaje: Java es un
lenguaje de programación compilado, lo que significa que el código fuente se compila
en un código de bytes que se ejecuta en la Máquina Virtual de Java (JVM). JavaScript,
por otro lado, es un lenguaje de script interpretado, lo que significa que
el código fuente se ejecuta directamente por el navegador web o por un entorno
de ejecución como Node.js.

Caracteristicas

  • Lenguaje de Alto Nivel
  • interpretado
  • Dinamico
  • Deblmente Tipado
  • Multi paradigma
  • Sensible a MAYUSCULAS y minusculas
  • No necesitas los puntos y comas al final de cada linea
